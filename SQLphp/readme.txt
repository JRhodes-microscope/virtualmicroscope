This directory contains the PHP files that are addressed by calls to the server from the user's instance of the the viewer and that return data to the viewer.
These files need to be modified to adapt to your institutions implementation of SQL.
The files should be located in a directory named "SQLphp" that is at the same level as the directory containing the HTML & javascript files (i.e., they are accessed by Ajax calls to "../SQLphp/'filename.php'.
